# 🌐 React Portfolio

Portfolio Website ที่สร้างด้วย React เพื่อแสดงผลงาน ทักษะ และข้อมูลส่วนตัวของผม

---> https://wonderful-empanada-153d22.netlify.app/

---

## 🚀 Features

* แสดงข้อมูลส่วนตัว (About Me)
* แสดงผลงาน (Projects)
* แสดงทักษะ (Skills)
* Responsive รองรับมือถือและเดสก์ท็อป
* โหลดเร็วด้วย Vite
* UI เรียบง่าย เน้นใช้งานจริง

---

## 🛠 Tech Stack

* React
* Vite
* CSS
* JavaScript

## ⚙️ Installation

```bash
git clone https://github.com/Testerzopa/react-portfolio.git
cd react-portfolio
npm install
npm run dev
```
